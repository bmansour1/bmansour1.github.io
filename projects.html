Project 2 - Automated WhatsApp Messenger

The Automated WhatsApp Messenger is a time-saving tool I developed to efficiently remind and schedule tennis lessons with my clients. Built using the PyWhatKit library in Python, this program automates the process of sending personalized WhatsApp messages to individual clients or groups. By providing the necessary details such as lesson timings, locations, and other relevant information, the automated messenger ensures that my clients receive timely reminders and updates without manual intervention. This streamlined communication process not only enhances the overall client experience but also allows me to focus more on delivering high-quality tennis lessons.

Project 3 - Tic Tac Toe Bot

The Python Tic Tac Toe AI Bot project is an interactive command-line game where players can challenge an intelligent AI opponent. The AI utilizes the Minimax algorithm with Alpha-Beta Pruning to make strategic moves, providing a challenging gaming experience. Players can choose to play as 'X' or 'O' against the AI, and the program ensures optimal decision-making for the AI, creating situations where it can either win or tie.
